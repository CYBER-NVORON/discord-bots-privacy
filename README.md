# How to publish with GitHub Pages

1. Create a public GitHub repository, for example: `discord-bots-privacy`.
2. Upload all `.md` files from this folder.
3. Open repository Settings → Pages.
4. Source: Deploy from a branch.
5. Branch: `main`.
6. Folder: `/root`.
7. Save.

URLs will look like:

- `https://YOUR_GITHUB_USERNAME.github.io/discord-bots-privacy/master`
- `https://YOUR_GITHUB_USERNAME.github.io/discord-bots-privacy/bear`
- `https://YOUR_GITHUB_USERNAME.github.io/discord-bots-privacy/wolf`
- `https://YOUR_GITHUB_USERNAME.github.io/discord-bots-privacy/owl`
- `https://YOUR_GITHUB_USERNAME.github.io/discord-bots-privacy/alice`

Replace `[CONTACT DISCORD USERNAME OR EMAIL]` in every policy with a real contact before publishing.
