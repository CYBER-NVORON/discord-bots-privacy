# Wolf Bot Privacy Policy

Last updated: June 16, 2026

This Privacy Policy applies to the Wolf Discord bot.

## What the bot does

Wolf Bot provides games and entertainment commands, including dice, rock-paper-scissors, random number rolls, Russian roulette, number guessing, server invite messages, and an AI chat feature in a designated Discord channel.

## Data processed

The bot may process:

- Discord user IDs
- Discord usernames and server nicknames
- Discord role information for language-specific responses
- Command inputs
- Message content for the number guessing game
- Message content when users mention the bot in the AI chat channel
- Direct messages sent to the bot
- Message attachments sent to the bot in direct messages

## How data is used

Data is used only for:

- game commands
- language-specific command responses
- AI chat responses in the configured AI channel
- admin message-sending commands
- staff-only review of direct messages sent to the bot

## Data storage

The bot does not intentionally store game messages or AI chat messages in its own database.

Direct messages to the bot may be forwarded to private staff-only Discord channels for review.

## External services

When users interact with the AI chat feature, their message content may be sent to an external AI service to generate a response.

Message content is used for response generation only and is not used by us to train AI models.

## Data sharing

We do not sell user data.

Data is not shared with third parties except when required by law, necessary to operate the bot through Discord, or necessary to generate AI responses through the configured AI service.

## Message content and AI training

We do not use message content for machine learning or AI model training.

## Data deletion

Users may request deletion of stored staff-log data by contacting the server administration.

Contact: nvoron482@gmail.com

## Access and security

Access to staff logs is limited to authorized server staff.

## Changes

This policy may be updated when the bot functionality changes.
