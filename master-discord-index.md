# Privacy Policy

Last updated: June 16, 2026

This Discord bot is used for moderation and community management on Discord servers.

## Data We Collect

The bot may process Discord user IDs, usernames, server nicknames, roles, message content, attachments, links, deleted and edited messages, join/leave events, role changes, nickname changes, avatar changes, voice channel activity, and moderation records.

Moderation records may include user ID, moderator ID, reason, time, and penalty points.

## How We Use Data

We use this data only for server moderation, anti-spam protection, scam-link detection, verification, role management, tickets, user reports, moderation logs, and appeals.

## Data Storage

Moderation warning records may be stored in the bot database.

Message logs and ticket archives may be sent to private staff-only Discord channels.

We do not sell user data.

We do not use message content for machine learning or AI model training.

## Data Sharing

We do not share user data with third parties, except when required by law or necessary to operate the bot.

## Data Deletion

Users may request deletion of their stored moderation data by contacting the server administration.

Contact: nvoron on Discord

## Security

Access to moderation logs and stored data is limited to authorized server staff.

## Changes

This policy may be updated when the bot functionality changes.
