# Bear Bot Privacy Policy

Last updated: June 16, 2026

This Privacy Policy applies to the Bear Discord bot.

## What the bot does

Bear Bot is used for server activity, XP, levels, clan features, role purchases, XP transfers, fun commands, and community interaction commands.

It provides features such as XP gain from messages, XP transfers, admin XP grants, role shop purchases, casino-style XP commands, clan-related automation, and social commands such as hug, kiss, bite, punch, duel, and Macarena.

## Data processed

The bot may process:

- Discord user IDs
- Discord usernames and server nicknames
- Discord role information
- Message events for XP calculation
- Message content for specific trigger features, such as DM handling and reaction/rep triggers
- XP balances and XP transactions
- Role purchases
- Clan membership and clan leadership data
- Member leave and role update events

## How data is used

Data is used only for:

- XP and level features
- clan management
- role shop purchases
- user-to-user XP transfers
- community interaction commands
- admin XP management
- staff-only DM review when users message the bot

## Data storage

The bot may store XP, level, role-shop, and clan-related records in the bot database or local bot storage.

Direct messages to the bot may be forwarded to private staff-only Discord channels for review.

## Data sharing

We do not sell user data.

Data is not shared with third parties except when required by law or necessary to operate the bot through Discord.

## Message content and AI training

Message content is not used for machine learning or AI model training.

## Data deletion

Users may request deletion of stored XP, clan, or moderation-related data by contacting the server administration.

Contact: nvoron482@gmail.com

## Access and security

Access to staff logs and stored bot data is limited to authorized server staff.

## Changes

This policy may be updated when the bot functionality changes.
